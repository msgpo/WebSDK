About this Release

This is a beta release of the Sony Ericsson Web SDK. This means that functionality will change before final release.

Limitations are listed in Release Notes

Introduction

Sony Ericsson is providing a Web SDK based on the open source PhoneGap framework, enabling developers to write native applications using HTML and JavaScript as an alternative to Java or C++.

The Web SDK supports

  • Sony Ericsson Android Xperia X10 and Symbian foundation based Satio phone.

Prerequisites

  • Java 6 SDK is required for the Web SDK to work properly.
  • To generate Android Web Applications, the Android SDK needs to be installed.

Installation

To install the Web SDK start by downloading it from Sony Ericsson Developer World.

The installation procedure should be self-explanatory.

The installation will install the following:

  • The Sony Ericsson Web Application Generator
  • Documentation, including this Getting Started document

Using the Web Application Generator

The Web Application Generator takes a folder anywhere on your disk and generates an application for the selected target.

Assume you have added all your asset files in the folder My App. All you need to do is right-click on the folder and select the correct target.

If you choose to generate Android application, the following dialog will pop up.

Just click on the "Make application" button. This will generate one signed and one unsigned APK-file. The signed APK-file is signed with the Android apkbuilder's default key.

The APK files will for now be generated in the asset directory so don't forget to remove the files before you regenerate the app

To run the application, upload it to the Android device or emulator.
