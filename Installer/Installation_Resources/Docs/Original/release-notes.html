Release Notes

November 13, 2009

This is a beta release of the Sony Ericsson Web SDK.

Please try out this release, but expect it to not work perfectly and maybe even be a bit inconsistent.

Use the appropriate forum at Sony Ericsson Developer World for feedback.

Latest release of this tool is available at Sony Ericsson Developer World

Supported APIs

  • Sony Ericsson Android phones support all APIs listed under Android on the PhoneGap site.
  • Sony Ericsson Symbian phones support all APIs listed under Symbian on the PhoneGap site.

Limitations

This release has a few limitations worth noting, following below

Web Application Generator

  • The tool uses the directory where the tool is installed as temp directory. If this becomes a problem, try to install the Web SDK into another directory
  • The tool has only been tested in a Windows XP environment
  • The output files e.g. APK or JAR files will be generated in the asset folder. This will be fixed.
