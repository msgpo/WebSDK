Release Notes

February 24, 2010

This is 1.2 release of the Sony Ericsson Web SDK.
It is based on Phonegap version 0.8.0.

What new:

·    New easier setup – Wizard that guides the user through the necessary settings and also helps to download JDK, Android SDK and Eclipse

·    New and better user interface. (Sony Ericsson look and feel).

·    Signing support for Android.

·    Button to launch the PhoneGap simulator.

·    Button to launch the Android SDK and AVD Manager

·    Tree view – Gives the user a better overview of his/her project folder.

·    Drag n drop support for project folder.

·    Eclipse plugin:
- Create web app projects.
- Build them with the WebSDK.
- Preview them in the simulator.

Use the appropriate forum at Sony Ericsson Developer World for feedback.

Latest release of this tool is available at Sony Ericsson Developer World

Supported APIs

  • Sony Ericsson Android and Symbian phones support all APIs listed under Android on the PhoneGap site.

Limitations

This release has a few limitations worth noting, following below

Web Application Generator

·    The tool has only been tested in a Windows XP environment

·    The tool support only Android 1.6 platform for the moment

·    The java JDK that should be used its Java 6 SDK
