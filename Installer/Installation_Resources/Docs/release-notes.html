Release Notes

August 9, 2010

 

This is 1.2.1 release of the Sony Ericsson Web SDK Packager based on PhoneGap framework version 0.9.1. WebSDK Packeger has Windows and Mac OS X versions.

 

What new:

  1. Path to PhoneGap Simulator, several versions of simulator could be used wth WebSDK Packager;
  2. Path to Android Project Stub, several versions of PhoneGap framework could be used with WebSDK Packager;
  3. New supported devices: Sony Ericsson Vivaz and Sony Ericsson Xperia X10 Mini, Sony Ericsson Xperia X8;
  4. Default zoom-out for small screen sizes (Xperia X10 Mini);
  5. MacOS X version

 

Use the appropriate forum at Sony Ericsson Developer World for feedback.

Latest release of this tool is available at Sony Ericsson Developer World

Supported APIs

Sony Ericsson Android and Symbian phones support all APIs listed under Android on the PhoneGap site.

Limitations

This release has a few limitations worth noting, following below

Web Application Generator

  1. The tool has only been tested in a Windows XP environment;
  2. The tool support Android targets 1.6 - 2.2;
  3. The JDK that should be used its Java 6 SDK.
