Installing the Skin for Xperia X10

Setting up an Xperia Skin

  1. Unzip the file in the add-ons directory of the Android SDK. This directory will contain the Google APIs, and a README.txt file

  2. Start Eclipse, and click on the phone button, or run the Android Command

  3. Click New, and fill out the fields. Note: The name of an Android Virtual Device can't contain any spaces, dashes or periods. Select Xperia as the target device. Once you are done, click "Create AVD".

  4. Start the AVD you have created. This AVD requires a high resolution monitor, so if you do not have one, it is a good idea to scale the size down when starting it.
